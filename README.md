# 🚀 AI Resume Assistant

An AI-powered Resume ATS Analyzer that helps job seekers optimize their resumes by comparing them with job descriptions and providing intelligent improvement suggestions.

## ✨ Features

* 🔐 Secure JWT Authentication (Login & Register)
* 📄 Upload Resume (PDF)
* 🤖 AI-powered Resume vs Job Description Analysis
* 📊 ATS Compatibility Score
* ✅ Matched Skills Detection
* ❌ Missing Skills Identification
* 💡 AI-generated Resume Improvement Suggestions
* 📜 Resume Analysis History
* 📑 View Individual ATS Reports
* 🗑️ Delete Previous Reports
* 📈 Interactive Dashboard

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router
* Tailwind CSS

* Axios

### Backend

* FastAPI
* SQLAlchemy
* PostgreSQL
* JWT Authentication
* Groq API (LLM)
* PyMuPDF (PDF Text Extraction)

## 📂 Project Workflow

1. User registers or logs in.
2. Uploads a resume in PDF format.
3. Enters the target job description.
4. AI analyzes the resume against the job description.
5. ATS score, matched skills, missing skills, and improvement suggestions are generated.
6. Reports are securely stored in PostgreSQL.
7. Users can revisit or delete previous ATS reports anytime.
